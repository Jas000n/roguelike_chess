## 2. 新需求方向（已确认）

1. **遗物系统改为海克斯系统**（战后/节点提供海克斯三选一）
2. **地图先不做分叉**，采用线性章节推进
3. **棋子扩充以变体为主**：
   - 车：坦克车、跑车等
   - 马：不同定位马（突进马、辅助马等）
   - 炮：导弹炮、迫击炮等
   - 通过“职业/阵营标签”来凑羁绊

## 3. 与目标成品相比，还缺哪些功能

## 3.1 自走棋核心层（TFT）

### A. 羁绊系统（最高优先级）
- 棋子标签体系：职业 + 阵营
- 2/4/6 分段羁绊逻辑
- 羁绊UI：已激活、差几只、效果说明

### B. 完整经济系统
- 利息（10/20/30/40/50）
- 连胜/连败奖励
- 经验与人口等级（决定上阵上限）
- 商店概率池（1~5费）

### C. 战斗与技能系统
- 主动技能（法力/冷却）
- 状态系统（灼烧/减速/护盾/破甲）
- 更稳定AI（选目标/技能释放优先级）

---

## 3.2 海克斯构筑层（替代遗物）

### A. 海克斯系统
- 每阶段给 3 选 1
- 稀有度：白/蓝/金（可后续加棱彩）
- 分类：经济类 / 战斗类 / 羁绊类 / 规则改写类

### B. 海克斯效果框架
- 全局增益（例如全队+攻速）
- 特定职业强化（例如“炮系伤害穿透”）
- 规则改写（例如“前2回合免控”）
- 叠加与冲突处理规则

### C. 线性流程中的海克斯触发点
- 建议：第1、3、5、7关后给海克斯
- Boss前给一次高质量海克斯

## 4. 棋子扩充方案（按你给的方向）

## 4.1 扩充原则
- 先扩“车/马/炮”三大类，每类做 3~4 变体
- 每个变体有清晰定位（坦克/输出/功能）
- 每个变体挂职业/阵营标签，服务羁绊构筑

## 4.2 示例设计（v0.2首批）

### 车系（前排）
- **坦克车**：高生命，高护甲，嘲讽
- **跑车**：高移速，高突进，切后排
- **震荡车**：近战AOE，附带击退/减速

### 马系（机动）
- **突袭马**：首击高爆发
- **战旗马**：邻近友军加攻速
- **梦魇马**：带恐惧/吸血

### 炮系（后排）
- **导弹炮**：单体高爆发，锁定远端
- **迫击炮**：范围轰炸，延迟落点
- **连发炮**：低单发高频率，持续输出

## 4.3 羁绊示例
- **钢铁系（2/4）**：前排减伤/反伤
- **机动系（2/4）**：移速/首击加成
- **火力系（2/4）**：炮系伤害与穿透提升
