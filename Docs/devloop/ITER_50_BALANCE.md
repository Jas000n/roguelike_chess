# 50轮玩法迭代（构筑与羁绊可成型性）

## 目标
- 把游戏从“能跑”推进到“更好玩”：阵容有易有难，羁绊可达，构筑有取舍。
- 做真机回归：每次 `B` 触发 `DevRunBalanceIterations(50)`，输出平衡数据。

## 本轮核心改动
1. 扩充棋子池，解决羁绊凑不出问题
- 新增 Artillery: `cannon_sniper`, `cannon_arc`, `cannon_scout`
- 新增 Assassin: `guard_poison`, `guard_mirror`
- 新增 Vanguard: `chariot_bulwark`, `chariot_ram`, `soldier_phalanx`
- 结果：主羁绊（先锋/炮/刺）均具备 4 层理论可达条件。

2. 阵容设计（易成型/难成型）
- 易成型：`钢铁炮阵`（先锋+炮）
- 易成型：`骑炮速攻`（骑兵+炮）
- 难成型：`四刺爆发`（4刺客+骑兵补位）
- 高难终局：`双四终局`（4先锋+4炮）

3. 自动平衡回归工具
- 新增 `DevRunBalanceIterations(int rounds)`，支持 50 轮自动构筑+战斗+奖励+海克斯。
- 支持快捷键：`B`（或 `F5`）触发 50 轮。
- 每次运行写入：
  - `~/Library/Application Support/DefaultCompany/DragonChessLegends/DevReports/balance_50_report.log`

4. 体验向调整
- 初始生命从 30 提升至 36。
- 后期关卡强度小幅下调（第6~8关 power: 4/5/6）。
- 准备阶段增加阵容建议文本，降低上手门槛。
