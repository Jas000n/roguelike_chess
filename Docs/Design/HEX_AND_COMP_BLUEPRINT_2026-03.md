# 海克斯与羁绊改版蓝图（2026-03）

## 目标
- 海克斯不只是数值补丁，而是能直接支撑一套玩法。
- 阵容分层清晰：有易成型的过渡阵，也有高上限终局阵。
- 羁绊数量与棋子池匹配，避免“理论存在但实战凑不出”。

## 参考到的赛季设计信号（用于抽象，不做照搬）
- 本赛季强调“对局内规则变化”与“选择质量”，单次选择可以改变运营路线。
- 对高经济海克斯做收敛，鼓励战力与节奏博弈，而不是单纯屯钱。
- 高阶终局阵需要更高条件，不应被过早无脑达成。

## 海克斯设计框架

### 1) 经济/运营型（保底但不失控）
- 金币雨：每回合 +4 金币。
- 理财大师：利息上限 +2。
- 精密改造：每回合额外 2 次免费刷新，并补 1 金币。
- 快速练兵：每回合额外经验。

### 2) 路线锚点型（单海克斯能开流派）
- 暗影契约：刺客爆发+斩杀滚雪球（参与击杀产金币）。
- 壁垒军令：先锋开战护盾+更高减伤，直接定义前排流。
- 火控超频：炮手第四次攻击附带溅射，直接定义炮手主C。
- 连环冲阵：骑兵节奏强化，首段爆发更稳定。
- 磐石誓约：石系肉度显著提升，走慢推/磨血线。
- 毒蚀载荷：毒系命中附加持续压血线，走处决链。
- 疾风步：风系速度+闪避，走先手与高频输出。

### 3) 通用战斗型（补短板）
- 全军增幅、战备修复、嗜血核心、处决边缘、超载部署。

## 12套阵容套路（按成型难度分层）

### 易成型
1. 钢铁炮阵（Vanguard + Artillery）
- 核心：坦克车/壁垒车 + 连发炮/电弧炮
- 核心海克斯：炮火专精、钢铁壁垒、火控超频

2. 骑炮速攻（Rider + Artillery）
- 核心：突袭马/跑车 + 连发炮
- 核心海克斯：连环冲阵、超远校准

3. 钢骑壁垒（Vanguard + Rider）
- 核心：壁垒车/枪骑马/突袭马
- 核心海克斯：壁垒军令、骑兵冲锋

4. 岩壁要塞（Vanguard + Guardian）
- 核心：岩石巨像 + 岩石前排 + 迫击炮
- 核心海克斯：磐石誓约、战备修复

### 中速成型
5. 毒影斩首（Assassin + Artillery）
- 核心：毒牙士/镜影士 + 狙击炮
- 核心海克斯：毒蚀载荷、处决边缘

6. 暗夜刺击（Assassin + Rider）
- 核心：暗影士/夜刃士 + 梦魇马
- 核心海克斯：刺影绽放、暗影契约

7. 疾风狙杀（Artillery + Rider, Wind/Shadow）
- 核心：跑车 + 侦察炮 + 狙击炮
- 核心海克斯：疾风步、火控超频

8. 烈焰突围（Vanguard + Artillery, Blaze）
- 核心：冲城车 + 导弹炮 + 火焰君主
- 核心海克斯：炮火专精、全军增幅

### 高难/高上限
9. 双四终局（4 Vanguard + 4 Artillery）
- 核心：高人口+经济管理
- 核心海克斯：超载部署、精密改造

10. 钢铁追三（Vanguard + Soldier）
- 核心：低费钢铁单位追3星
- 核心海克斯：追三计划、精密改造

11. 毒蚀处决（Venom + Assassin/Rider）
- 核心：毒系压血线 + 刺客收割
- 核心海克斯：毒蚀载荷、暗影契约

12. 经济转终局（前期经济海克斯，后期转双四/刺客）
- 核心：金币雨/理财大师开局，7-8级转高费质量
- 核心海克斯：金币雨、理财大师、超载部署

## 迭代检查清单
- 每局至少出现 1 个“路线锚点海克斯”。
- 2-1 到 4-2 阶段，易成型阵容胜率应有可感知优势。
- 终局阵容需要运营门槛，但不是纯运气才能达成。
- 羁绊面板中，常用阵营都应在 2 人阈值可激活。
