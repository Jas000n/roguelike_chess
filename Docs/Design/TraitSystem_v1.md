# Trait System v1

## 目标

- 维护当前版本的职业、阵营、棋型羁绊设计
- 记录哪些兵种承担前排、后排、切入、治疗职责
- 避免出现“某个家族/职业全部做同一件事”的单一化问题

## 当前职业分工

### Vanguard

- 角色：主前排、承伤
- 代表：坦克车、震荡车、壁垒车、方阵兵、护卫兵

### Rider

- 角色：机动切入、节奏启动
- 代表：突袭马、疾风马、枪骑马
- 补充：战旗马拆到 Guardian，梦魇马拆到 Assassin，避免“马=全员骑兵”

### Artillery

- 角色：后排持续输出、溅射
- 代表：导弹炮、迫击炮、连发炮、狙击炮

### Controller

- 角色：功能炮、控场追加伤害、远程压制
- 代表：电弧炮、侦察炮、寒晶炮、天幕炮
- 说明：这条职业是为了把“炮”家族从纯输出里拆开，保留家族大羁绊，同时让炮内部出现输出炮与功能炮两条线

### Assassin

- 角色：开战切后排、斩杀脆皮
- 代表：暗影士、毒牙士、雾刃士
- 机制：2/4 刺客时开战直接切入敌方后排附近

### Guardian

- 角色：副前排、团队护卫
- 代表：岩石巨像、镜影士、震荡车、战旗马

### Soldier

- 角色：稳定站场、持续普攻
- 代表：剑士兵、夜刃士

### Medic

- 角色：治疗、维持前排血线
- 代表：圣卫士、誓约兵、玄灵象
- 机制：2/4 医者时行动会为最低生命友军回复生命

### Leader

- 角色：单卡核心、团队光环
- 代表：火焰君主

## 当前“士”家族分工审视

过去版本的问题：

- 几乎所有“士”都被塞进 Assassin
- 结果是“士”家族内部没有层次，玩家看到“士”就默认是切后排
- 这不符合象棋题材的角色差异，也让阵容构筑太扁平

现在的拆分：

- 暗影士：Assassin / Shadow
- 夜刃士：Soldier / Night
- 毒牙士：Assassin / Venom
- 镜影士：Guardian / Mist
- 雾刃士：Assassin / Mist
- 圣卫士：Medic / Holy

结论：

- “士”不再只承担刺客职能
- 同一家族内部出现切入、站场、护卫、治疗四种职责
- 后续可继续补一张 5 费“士”系核心，承担残局收割或团队指挥

## 家族不是职业

本项目后续遵循这个原则：

- 家族负责“中国象棋味”和大羁绊
- 职业负责战斗职责和玩法差异

例子：

- 炮家族：
  - 大羁绊依然吃 `马后炮 / 车炮联营`
  - 但职业内部拆成 `Artillery` 和 `Controller`
- 马家族：
  - 依然能吃 `马后炮`
  - 但职业内部拆成 `Rider / Guardian / Assassin`
- 车家族：
  - 依然能吃 `车兵推进 / 车炮联营`
  - 但职业内部拆成 `Vanguard / Guardian / Assassin`

这样既保留“大羁绊成型爽感”，又避免“整家族只有一个按钮”的单一化

## 阵营职责建议

- Steel：硬前排、通用减伤
- Stone：慢节奏厚前排
- Blaze：主伤害放大
- Shadow：爆发与斩杀
- Night：开局爆发
- Thunder：连锁追加伤害
- Frost：破坚与溅射
- Wind：机动、闪避
- Venom：持续伤害
- Mist：规避、扰乱
- Holy：治疗与续航

## 当前存在的问题

- Medic 目前数量仍少，2/4 医者能成型但分支还不够丰富
- Guardian 与 Vanguard 的前排差异还可以继续拉开
- Leader 只有 1 张，后续应补另一个高费领袖或副领袖
- Soldier 体系目前更像“稳定补充”，还缺真正的中后期主 C
- Controller 目前数量刚够成型，后续还需要补第 4 张高质量控场单位
- 现已补入 5 费 Controller 核心：天幕炮
- 现已补入 5 费 Medic 核心：玄灵象

## 后续扩展方向

1. 增加一张高费 Medic 或 Holy 单位，支撑纯续航体系
2. 增加一张 Soldier 主 C，形成兵系追三路线
3. 给 Guardian 增加反击/援护机制，和 Vanguard 的纯减伤区分开
4. 增加更多家族交叉棋型，让“士象全”“车兵推进”之外有更多中国象棋味组合
